# Atharva Portfolio

Static portfolio website built for GitHub Pages.

## Files
- `index.html`
- `style.css`
- `script.js`
- `.nojekyll`

## Deploy on GitHub Pages
1. Create a public GitHub repository.
2. Upload all files from this folder to the repository root.
3. Go to **Settings > Pages**.
4. Set source to **Deploy from branch**.
5. Choose **main** branch and **/(root)** folder.
6. Save and wait for the Pages link.

## Notes
- This is a plain HTML/CSS/JS portfolio.
- No build step is required.
- All paths are relative, so it works on GitHub Pages.
